# MediCare - Healthcare Medicine Delivery Web App

A modern, professional navigation bar and multi-page website for a healthcare medicine delivery application.

## Features

- **Modern Navigation Bar**: Dark theme with glassmorphism effects, soft shadows, and rounded elements
- **Responsive Design**: Works on desktop and mobile devices
- **Active Page Highlighting**: Automatically highlights the current page in the navigation
- **Cart Badge**: Displays item count in the cart (demo: shows 3)
- **Admin Button**: Prominent gradient-styled button for admin access
- **Smooth Animations**: Hover effects and transitions for a premium feel

## Pages

- **Home** (`index.html`): Welcome page with hero section
- **Medicines** (`medicines.html`): Browse medicines
- **Cart** (`cart.html`): View cart items
- **My Orders** (`myorders.html`): Order history
- **Contact** (`contact.html`): Contact information

## Technologies Used

- HTML5
- CSS3 (with backdrop-filter for glassmorphism)
- JavaScript (ES6+)

## Color Palette

- Background: Dark blue/black gradient (#1a1a2e to #16213e)
- Text: White (#ffffff) and soft gray (#b0b0b0)
- Accent: Orange for notifications (#ff6b35), Blue for primary actions (#007bff)

## How to Run

1. Open `index.html` in your web browser
2. Navigate between pages using the menu items
3. The active page will be highlighted in the navigation bar

## Browser Support

- Modern browsers with support for CSS `backdrop-filter`
- Fallback for older browsers may not show glassmorphism effect

## Customization

- Modify colors in `css/style.css`
- Update content in the respective HTML files
- Adjust JavaScript behavior in `js/script.js`
